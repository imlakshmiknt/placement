DEMO CHECKLIST
20-Minute Demo Structure (Allen/Aakash Standard)
0–2 min
Hook + Real Life Connect
Start with a question or real-life example. "Aapne kabhi lift mein heavier feel kiya? Kyun hota hai?"
2–6 min
Concept Explanation
Theory with board diagrams. Simple language. Layer 1 → Layer 2. No jargon initially.
6–14 min
Solved Example (JEE Level)
Step-by-step on board. Talk out loud while solving. "Pehle FBD banate hain..."
14–18 min
Student Interaction
Ask students to solve a similar problem. Address doubts. Show patience.
18–20 min
Summary + Next Class Preview
Recap 3 key points. "Agle class mein..." — shows planning ability to evaluators.
